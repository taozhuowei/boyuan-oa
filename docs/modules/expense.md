# biz.expense — 费用报销（待集成）

> 此模块尚未集成到当前基座。集成时在此补充完整设计文档。

## 计划功能

- 费用申请：员工提交报销申请，填写费用类型、金额、事由，附上票据扫描件
- 分级审批：按报销金额区间配置不同审批链（小额由部门经理审批，大额需 CEO/财务二级审批）
- 财务确认：财务角色在审批通过后执行"已打款"确认，变更单据状态
- 报销统计：按员工、部门、时间范围汇总报销金额，供财务出具费用报告

## 与基石模块的集成点

- `core.form` — 报销申请作为 formType=EXPENSE 的通用表单提交，进入待审批队列
- `core.approval` — 报销审批流由审批流引擎驱动，支持按金额区间绑定不同节点配置
- `core.file` — 票据扫描件通过 AttachmentController 上传，businessType 设为 EXPENSE，与报销单 ID 关联
