# biz.project — 项目管理（待集成）

> 此模块尚未集成到当前基座。集成时在此补充完整设计文档。

## 计划功能

- 项目立项：PROJECT_MANAGER 创建项目，填写项目名称、类型、起止日期、合同金额等基本信息
- 成员管理：为项目添加/移除成员，指定角色（项目经理/技术负责人/执行成员）
- 里程碑与进度：设置项目里程碑节点，记录完成状态和实际完成日期
- 收入支出记录：按项目记录收款节点和支出明细，汇总项目盈亏

## 与基石模块的集成点

- `core.employee` — 项目成员须从员工档案中选取；PROJECT_MANAGER 角色员工可管理本人负责项目的数据访问权限
- `core.form` — 项目相关的审批事项（如大额支出审批）通过通用表单流转，formType 为项目专属类型
- `core.file` — 项目合同文件、竣工资料等附件通过 AttachmentController 上传，businessType 设为 PROJECT，businessId 为项目 ID
