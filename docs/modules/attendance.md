# biz.attendance — 考勤管理（待集成）

> 此模块尚未集成到当前基座。集成时在此补充完整设计文档。

## 计划功能

- 请假申请：员工提交请假申请（年假/病假/事假/调休/补偿假等），触发对应审批流
- 加班申请：员工提交加班申请，经审批后记入加班档案，作为调休或加班费计算依据
- 考勤记录管理：记录每日出勤状态（正常/迟到/早退/缺勤），支持 HR/部门经理修正
- 假期余额管理：维护员工各类假期余额（年假天数、调休余额等），申请时校验余额充足

## 与基石模块的集成点

- `core.form` — 请假申请和加班申请通过通用表单流转机制提交，审批结果写回考勤记录
- `core.approval` — 请假和加班的多节点串行审批由审批流引擎驱动，支持按部门/项目配置不同审批链
- `core.employee` — 读取员工所属部门和直系上级，用于审批人解析；考勤记录以员工 ID 为主键关联
