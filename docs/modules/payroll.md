# biz.payroll — 薪资管理（待集成）

> 此模块尚未集成到当前基座。集成时在此补充完整设计文档。

## 计划功能

- 薪资周期管理：按发薪周期（payroll.payday 配置）生成当期薪资记录
- 工资条生成：汇总员工基础薪资、绩效系数、出勤天数、加班补贴、扣款项，生成工资条明细
- 薪资调整：记录薪资变更历史，支持追溯对比
- 薪资发放状态跟踪：标记已发放/待发放，支持批量确认
- 薪资报表导出：按部门或周期导出 Excel

## 与基石模块的集成点

- `core.employee` — 读取员工基础薪资覆盖（baseSalaryOverride）、绩效系数（performanceRatio）、合同类型等薪资参数
- `core.form` — 汇总当期已审批通过的加班申请，计算加班补贴；读取请假记录计算出勤扣款
- `core.notification` — 工资条生成后向员工发送通知；发薪日临近时提醒财务负责人
