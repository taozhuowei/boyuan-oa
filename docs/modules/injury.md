# biz.injury — 工伤补偿（待集成）

> 此模块尚未集成到当前基座。集成时在此补充完整设计文档。

## 计划功能

- 工伤申报：HR 或部门经理代工人填写工伤事故报告，记录受伤时间、地点、伤情描述
- 理赔记录管理：追踪工伤理赔进度（待审核/已上报/理赔中/已结案），记录赔付金额
- 医疗费用挂载：将工人提交的医疗发票附件关联到对应工伤记录
- 历史档案查询：HR/CEO 可按工人或时间范围查询工伤历史记录

## 与基石模块的集成点

- `core.form` — 工伤申报作为 formType=INJURY 的通用表单提交，经主管审批后转入理赔流程
- `core.employee` — 工伤记录以工人员工 ID 为关联键，查询工伤档案时校验员工类型（WORKER）
