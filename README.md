# 博渊 OA 工作台

企业微信工作台小程序 + Web 管理后台，面向建筑与工程企业的内部办公协同系统。

## 核心架构

```
┌─────────────┐    ┌─────────────┐
│   小程序端   │    │    H5 端    │
│ (uni-app)   │    │ (uni-app)   │
└──────┬──────┘    └──────┬──────┘
       │                  │
       └────────┬─────────┘
                │ HTTP / REST
                ▼
       ┌─────────────────┐
       │  Spring Boot    │
       │  PostgreSQL     │
       └─────────────────┘
```

## 目录结构

| 目录 | 说明 |
|------|------|
| [app/](./app/) | 前端应用 |
| [server/](./server/) | 后端服务 |
| [docs/](./docs/) | 技术文档 |
| [test/](./test/) | 测试套件 |

## 快速开始

```bash
# 安装依赖
yarn install

# 启动开发环境（后端+H5+小程序）
yarn dev

# 运行测试
test/run-all.bat     # Windows
bash test/run-all.sh # Unix/macOS

# 构建生产包
yarn build
```
