# OA 前端项目

## 技术栈

- uni-app 3.0
- Vue 3
- Vite
- TypeScript
- Pinia

## 目录结构

```
src/
├── components/     # 公共组件
├── pages/          # 页面
├── static/         # 静态资源
├── stores/         # Pinia 状态管理
├── utils/          # 工具函数
├── App.vue         # 应用入口
├── main.ts         # 主入口
└── pages.json      # 页面配置
```

## 启动命令

```bash
# Web 端开发
yarn dev:web

# 小程序开发
yarn dev:mp-weixin

# 小程序预览
yarn preview:mp-weixin

# 运行测试
yarn test:web
```
