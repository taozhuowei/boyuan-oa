# 博渊·建筑工程版 — 业务页面 UI 设计

> **文档职责**：定义建筑工程版所有业务模块的页面布局，包括工作台（各角色视图）、考勤管理、薪资管理、项目管理、员工管理、角色管理和数据管理。
>
> **目标读者**：前端开发者、UI 设计师。
>
> **不包含内容**：平台级 Shell（TopBar / LeftNav / 底部 Tab）、登录页、待办中心、系统管理控制台——见 `platform/UI_DESIGN.md`。

---

## 图例

- `[========]` — 容器 / 卡片区域
- `<Component>` — 组件库组件（AntD / Vant）
- `{Custom}` — 自定义业务组件
- `{{data}}` — 动态数据绑定
- `[R:角色]` — 角色可见标记

|          |  <Button>新增角色 [R:ceo]  |  角色名称: <Input>          |

|          |  │   项目经理 [系统内置]│ |  <Checkbox> 工伤补偿        |
|          |  │   CEO     [系统内置] │ |  <Checkbox> 施工日志填报    |
|          |  │   ─────────────── │ |  <Checkbox> 项目初审        |
|          |  │   {{自定义角色}}    │ |  <Checkbox> 查看全员信息    |
|          |  └──────────────────────┘ |  <Checkbox> 薪资结算        |

- **CEO**：可新增角色 / 删除自定义角色 / 配置"终审审批"权限
