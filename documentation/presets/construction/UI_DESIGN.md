# 博渊·建筑工程版 — 业务页面 UI 设计

> **文档职责**：定义建筑工程版所有业务角色在所有页面上的布局、可见内容、可操作项和所用组件。是前端开发的详细指导文档，覆盖每个角色的完整操作流程。
>
> **目标读者**：前端开发者。
>
> **不包含内容**：Shell（TopBar / LeftNav / 底部 Tab）、登录页、初始化向导、待办中心、通知中心、个人档案——见 `platform/UI_DESIGN.md`；组件开发状态——见 `dev/TODO.md`。

---

## 图例

- `[========]` — 容器 / 卡片区域
- `<Component>` — 组件库组件（H5: AntD / MP: Vant）
- `{Custom}` — 自定义业务组件
- `{{data}}` — 动态数据绑定
- `[R:角色]` — 仅指定角色可见或可操作
