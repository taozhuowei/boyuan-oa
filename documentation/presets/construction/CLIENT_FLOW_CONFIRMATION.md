# 博渊·建筑工程版 — 业务流程确认文档

> **用途**：提交甲方确认核心业务流程及关键规则，请逐项阅读并反馈。
>
> **阅读说明**：本文档不含系统技术术语，仅描述实际操作流程和业务规则。

---

## 一、考勤申请流程

### 请假申请
1. 员工在系统中填写请假类型（年假/事假/病假/婚假/产假/其他）、起止日期及原因，可附上证明材料（图片或 PDF）
2. 提交后，所属项目的**项目经理**进行初步审核（通过 / 驳回）
3. 项目经理通过后，由 **CEO** 进行最终审批
4. CEO 审批通过后请假单归档，系统自动将请假天数纳入当月工资计算（按类型扣款，病假按半日薪扣除）
5. 任一环节驳回后，员工可查看驳回原因，并重新提交申请

> **若所属项目当前没有项目经理**，系统自动将初审转给 CEO 处理。

### 加班申请
流程与请假申请相同，加班记录用于计算当月加班补贴（平日 1.5 倍、周末 2 倍、法定节假日 3 倍时薪）。

> **最小计量单位**：请假和加班的最小计算精度（小时 / 半天 / 天）可在系统初始化时统一设定。

## 二、工伤补偿流程

### 申请提交
- 受伤员工本人、或任何同事均可代为提交工伤申请，填写受伤日期、受伤部位、情况描述，并上传医疗单据
- 代他人提交时，需填写被代提交人的工号

### 审批路径
| 提交人          | 审批流程                               |
|----------------|--------------------------------------|
| 员工本人         | 项目经理初审 → CEO 终审                |
| 项目经理代为提交  | 直接由 CEO 终审（跳过初审）              |
| 其他同事代为提交  | 项目经理初审 → CEO 终审                |

### 理赔金额录入
- 工伤审批完成后，**保险理赔需要一段时间**
- 理赔完成后，由**财务**录入最终理赔金额，并指定将该金额计入**哪个月的工资单**
- 因此，理赔金额入账的月份可以与工伤发生月份不同，系统不强制同月结算

## 三、施工日志流程

> **施工日志是独立系统**，与薪资无关，不影响工资发放。

### 提交
- 劳工每天在系统中填写当日施工日志
- 填写方式：在表单中添加若干行工作项（工作内容 + 数量 + 单位），例如：
  - 焊接口 / 8 / 个
  - 管道铺设 / 24 / 米
- 每行数据自由添加，无需事先配置模板；工作项之外可填写补充说明
- 系统自动将工作项汇总为表格和文字描述，形成最终日志

### 审核
- 日志提交后，由所属项目的**项目经理**审核（通过即归档）
- 驳回后劳工可修改重新提交

### CEO 追溯驳回
- CEO 可在**任何时间**对已归档的施工日志发起驳回，要求重写
- 此操作会通知劳工，劳工须重新提交；原日志记录完整保留

## 四、薪资结算流程

### 结算周期配置
以下参数均可由管理员配置：

| 参数         | 说明                         |
|------------|------------------------------|
| 结算周期     | 月结 或 半月结                 |
| 起止日       | 每个周期的开始和结束日期         |
| 发薪日       | 支持次月发薪（如次月10日）       |
| 结算截止日   | 财务须完成结算的日期             |
| 提醒时间     | 结算截止日前 7 天，系统提醒财务   |

### 结算步骤
1. 财务发起**预结算**，系统自动校验以下 4 项（全部通过才可正式结算）：
   - 所有员工考勤数据已录入，无未处理异常
   - 所有请假 / 加班单据审批已完成
   - 所有工伤补偿申请审批已完成
   - 所有在职员工薪资档案已配置

2. 校验通过后，财务执行**正式结算**，系统生成每位员工的工资单

3. 工资单发布后，员工可查看详情并**电子签名确认**（需提前完成签名绑定和 PIN 码设置）

### 工资组成
- 基本工资（按岗位档位取值）
- 加班补贴（自动计算）
- 工伤补偿（财务手动录入，可跨月）
- 绩效奖金（财务手动录入，仅适用于办公室员工）
- 请假扣款（自动计算）
- 社保（个人部分）
- 住房公积金（个人部分）
- 个人所得税
- **实发工资** = 以上各项汇总

> **社保处理方式**（二选一，初始化时配置）：
> - **公司代缴**：社保由公司代为缴纳，工资条显示各险种个人部分的扣除金额
> - **并入工资**：五险一金补贴以合计金额并入工资发放，工资条只显示"五险一金补贴（合计）"一行，不分险种明细；员工须自行前往社保机构办理**灵活就业**参保手续，公司不代缴

### 预结算例外处理
- 预结算时，若有少量请假/加班/工伤单据尚未完成审批，财务可提交**例外申请**，说明原因
- CEO 审批通过后，系统允许跳过对该批单据的检查，继续结算；操作记录永久保留
- **考勤数据缺失**和**员工档案未配置**这两项不支持例外处理，必须补齐才能结算

### 薪资更正
- 正式结算后如需修改，由**财务发起更正申请**，填写原因，提交 CEO 审批
- CEO 批准后解锁周期，财务重新结算，生成新版本工资单
- 历史版本完整保留，员工重新签名确认最新版本

## 五、数据管理

- 所有业务数据默认保留 **1 年**
- 到期前 **30 天**，系统自动提醒 CEO：数据类型、预计到期日期、数据数量
- CEO 可选择：
  - **导出后删除**：系统自动打包导出，完成后执行清理
  - **忽略**：到期后系统自动清理，不再提醒
- **审批操作记录**永久保留，不受1年限制影响

## 六、权限说明

| 角色       | 主要职责                                             |
|----------|-----------------------------------------------------|
| CEO      | 最终审批（考勤/工伤）；薪资更正审批；施工日志追溯驳回；全局配置   |
| 财务     | 薪资结算；工伤理赔录入；员工档案协管                         |
| 项目经理  | 初步审批（考勤/工伤）；施工日志审批；管理本项目成员              |
| 员工     | 发起本人申请；查看本人工资条；为他人代提交工伤申请               |
| 劳工     | 提交施工日志；发起工伤补偿申请；查看本人工资条                   |
