# 博渊·建筑工程版 — 业务需求设计

> **文档职责**：描述建筑工程版预置方案的企业背景、业务场景、启用模块和用户故事。这是定制化的业务需求，而非平台通用能力。
>
> **目标读者**：产品经理、业务顾问、前端/后端开发者（了解业务背景）。
>
> **不包含内容**：角色权限配置（见 `ROLE_CONFIG.md`）；工作流与薪资规则（见 `WORKFLOW_CONFIG.md`）；页面布局（见 `UI_DESIGN.md`）。

---

## 1. 企业背景

**典型客户画像**：中小型建筑与工程企业，员工规模 50～500 人。

**核心业务特点：**
- 以项目制为核心组织形式，一个项目对应一个工地，由项目经理负责
- 员工分为**办公室员工**（行政、财务、管理）和**劳务工人**（现场施工人员）两类，薪资结构和权限差异显著
- 工资结算依赖考勤、请假、加班、工伤补偿等多项数据，人工核算易出错
- 施工日志是项目履约的重要记录，需要日报 + 审批留存
- 工资条确认存在法律风险，需要电子签名存证

**核心痛点：**
- 请假/加班流程靠微信群和纸质审批，历史记录难以追溯
- 每月工资手工计算，数据散落在多个 Excel 文件
- 劳工工伤补偿申请无统一入口，处理周期长
- 施工日志填报不规范，项目经理审批滞后

## 2. 启用的业务模块

建筑工程版启用以下模块（通过系统配置控制）：

| 模块 | 启用 | 说明 |
|------|------|------|
| 考勤管理（请假/加班） | ✓ | 所有员工可发起，支持两级审批 |
| 工伤补偿 | ✓ | 劳工专属，支持代录入 |
| 施工日志 | ✓ | 劳工填报，项目经理审批（单节点） |
| 薪资管理 | ✓ | 财务执行结算，员工签名确认 |
| 员工管理 | ✓ | CEO / 财务维护档案，分配角色 |
| 角色管理 | ✓ | CEO / 财务配置权限，支持自定义角色 |
| 数据生命周期管理 | ✓ | CEO 配置数据保留策略 |
| 企业微信通讯录导入 | ✓（预留）| 财务 / CEO 触发，当前使用 mock 数据 |

## 3. 业务场景说明

### 3.1 考勤申请

**请假场景：**
- 员工发起请假申请，填写类型（年假/事假/病假/婚假/产假）、时间段、原因
- 系统自动计算请假天数及对应扣款
- 流转至项目经理初审 → CEO 终审
- 通过后自动影响当月薪资结算

**加班场景：**
- 员工自行发起，或项目经理代为发起
- 填写加班时长、加班类型（平日/周末/节假日）
- 系统自动计算加班补贴
- 与请假申请共用同一审批流

**考勤异常：**
- 每月结算前，财务需核对考勤数据，异常（未提交申请的缺勤）需在结算前处理

### 3.2 工伤补偿

- 仅劳务工人（`worker`）角色可见此入口
- 工人本人、财务、项目经理均可录入工伤信息
- 填写：受伤时间、部位、类型、医疗单据（附件）、补偿金额
- 补偿申请通过后，金额自动纳入当月薪资结算
- 工伤补偿数据受数据保留策略管理

### 3.3 施工日志

- 仅劳工角色可提交日志，每日一份
- 按项目经理配置的日志模板填写：工作内容、工时、现场照片（最多 5 张）
- 提交后进入项目经理审批（单节点，无 CEO 终审）
- 日志审批通过后自动归档，CEO / 项目经理可查看项目日志汇总

### 3.4 薪资结算

**结算流程：**
1. 财务在结算周期发起预结算（生成 PREVIEW 状态周期）
2. 系统执行 5 项完整性校验：
   - 考勤数据是否完整
   - 审批中的单据是否处理完毕
   - 施工日志是否审批完毕（劳工）
   - 工伤补偿是否处理完毕
   - 员工档案是否完整
3. 校验通过 → 正式结算，周期锁定
4. 员工登录查看工资条，电子签名确认（或发起异议）
5. 财务发起更正申请 → CEO 审批解锁 → 重新结算
6. 周期关闭 → 归档

**工资条电子签名：**
- 员工首次使用时在 Canvas 手写签名并设置独立 PIN 码
- 后续确认：展示签名预览 → 输入 PIN → 系统生成存证 PDF

### 3.5 员工管理

**员工档案包含：**
工号（自动生成）、姓名、手机号、部门、角色、员工类型（办公/劳务）、入职日期、所属项目

**通讯录导入：**
- 从企业微信批量导入，支持预览映射、增量更新
- 导入时自动生成工号和初始密码

### 3.6 数据生命周期

| 数据类型 | 默认保留期 | 特殊说明 |
|---------|----------|---------|
| 请假 / 加班记录 | 5 年 | |
| 工伤补偿记录 | 永久 | 涉及法律合规 |
| 施工日志 | 3 年 | |
| 工资条及确认记录 | 劳动合同终止后 5 年 | 含存证 PDF |
| 电子签名档案 | 永久 | 不允许删除 |
| 审批历史 | 5 年 | |
