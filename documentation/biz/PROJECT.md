# 众维 OA 工作台 — 项目介绍

## 1. 项目概述

众维 OA 工作台是面向建筑与工程企业的内部办公协同系统，提供**企业微信工作台小程序**与**Web 管理后台**双端入口，覆盖组织管理、项目协同、审批流转、薪资结算与数据留档。

## 2. 核心价值

- **统一移动入口**：员工通过企业微信工作台即可处理待办、查看工资条、提交业务单据
- **审批闭环**：请假、加班、工伤补偿、施工日志统一流转，初审+终审两级审批，状态自动归档
- **自动算薪**：基于考勤、请假、加班、工伤补偿结果自动计算工资，减少人工核算错误
- **数据留档与合规**：工资确认留痕（时间、IP、设备），业务历史与结算结果可追溯，支持审计
- **数据生命周期管理**：到期提醒、延期、备份导出、自动清理，满足企业数据治理要求

## 3. 功能模块

### 组织管理
- 部门与项目架构维护
- 员工档案管理
- 企业微信通讯录批量导入（预览、映射、增量更新）

### 业务中心
- **请假**：年假、事假、病假等类型，自动计算时长与扣款
- **加班**：员工发起或项目经理代发起，自动计算加班工资
- **考勤**：周期考勤数据维护，异常需在结算前处理
- **工伤补偿**：仅劳工可见，支持代录入，补偿金额纳入工资结算
- **施工日志**：仅劳工可提交，按项目模板填写，项目经理审批

### 审批中心
- 待办统一入口，按业务类型、状态、项目、申请人筛选
- 审批意见、附件查看、历史流转记录
- 审批通过后自动归档；驳回后支持重新发起

### 薪资中心
- 薪资规则配置（基本工资、绩效、加班、扣款、社保、公积金、个税）
- 预结算试算、正式结算、周期锁定
- 工资条查看、确认、异议发起
- 结算更正申请（财务发起 → CEO 审批 → 解锁重算）

### 数据中心
- 数据有效期配置（全局默认 + 按类别覆盖）
- 到期前主动提醒，支持延期或额外备份导出
- 到期自动清理，同步删除数据库记录与附件物理文件

## 4. 角色与权限

系统预置 4 个默认角色 + 1 个员工类型，CEO 可自定义扩展角色。

| 角色 | 核心权限 |
|---|---|
| **员工** | 发起请假/加班、查看并确认本人工资条 |
| **劳工** | 在员工基础上，可提交施工日志、发起工伤补偿 |
| **财务** | 维护人员与薪资配置、执行结算、导入通讯录、复核工资异议 |
| **项目经理** | 处理项目范围内初审、维护施工日志模板、查看项目总览 |
| **CEO** | 管理全局配置、终审审批、配置角色权限、查看经营总览、配置数据有效期 |

## 5. 终端支持

| 终端 | 定位 | 适配 |
|---|---|---|
| **企业微信工作台小程序** | 移动办公、高频审批、消息提醒、轻量录入 | 主流手机竖屏 |
| **Web 端** | 配置管理、批量处理、复杂查询、报表分析 | 手机浏览器 / 平板 / PC 宽屏 |
